# Qwentest Project — Конфигурация для Qwen Code

## Стек технологий
- **Python**: предпочтительные фреймворки и библиотеки
- **C#**: .NET 6+/8+, ASP.NET Core, консольные приложения

## Правила работы с кодом

### Python
- Использовать type hints (PEP 484)
- Следовать PEP 8 для форматирования
- Предпочитать f-строки для форматирования строк
- Использовать `async/await` для асинхронных операций
- Менеджеры пакетов: `pip`, `poetry` или `uv`

### C#
- Использовать C# 10+ синтаксис (global using, file-scoped namespaces)
- Следовать .NET coding conventions
- Использовать `async/await` для асинхронных операций
- Предпочитать dependency injection
- Использовать pattern matching где уместно

## Структура проекта
```
Qwentest/

```

## Инструменты разработки

### Python
- Линтер: `ruff` или `flake8`
- Форматтер: `black` или `ruff format`
- Тесты: `pytest`
- Type checker: `mypy`

### C#
- SDK: .NET 8+
- Форматтер: `dotnet format`
- Тесты: `xUnit` или `NUnit`
- Анализ: Roslyn analyzers

## Агент: правила использования

Использовать агентов для:
- Поиска по большой кодовой базе (Explore agent)
- Сложных многоступенчатых задач (General-purpose agent)
- Рефакторинга и тестирования
