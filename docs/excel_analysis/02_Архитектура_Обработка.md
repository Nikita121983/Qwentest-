## 3. Spill-формулы (динамические, авто-растягивающиеся)

### Список фильтр — центр spill-архитектуры

| Ячейка | Spill → | Формула | Назначение |
|--------|---------|---------|------------|
| G4 | G4:G243 | `=FILTER(F4:F1402,NOT(F4:F1402=""))` | Фильтр пустых |
| Q4 | Q4:Q15 | `=TRIM(SORT(UNIQUE(FILTER(INDIRECT($M$1),NOT(ISBLANK(INDIRECT($M$1)))))))` | Уникальные сортированные |
| T4 | T4:T6 | `=TRIM(SORT(UNIQUE(INDIRECT($N$1))))` | То же, другой диапазон |
| V4 | V4:V49 | `=TRIM(SORTBY(Q5:Q50,P5:P50))` | Сортировка по ключу |
| W4 | W4:W5 | `=TRIM(SORTBY(UNIQUE(INDIRECT($J$1)),UNIQUE(INDIRECT($J$1))))` | Уникальные по 2 ключам |
| X4 | X4:X25 | `=IFERROR(TRIM(SORTBY(UNIQUE(INDIRECT($L$1)),UNIQUE(INDIRECT($K$1))))...)` | Комбинированная сортировка |
| Z4 | Z4:Z24 | `=FILTER(X4:X202,(X4:X202)<>"")` | Фильтр непустых |
| AA4 | AA4:AA24 | `=TOCOL(TOCOL($Y$4:INDEX(...))&AA3&TOROW($Z$4:INDEX(...)))` | Декартово произведение |

| AG4 | AG4:AG25 | `=SORTBY(UNIQUE(INDIRECT($L$1)),UNIQUE(INDIRECT($K$1)))` | Сортировка уникальных |

| AO4 | AO4:AO1400 | `=TRIM(SORTBY(INDIRECT($G$1),INDIRECT($I$1),1,INDIRECT($K$1),1,...))` | Сортировка по 3+ ключам |

## 4. Динамические диапазоны (CONCAT → INDIRECT)

На «Список фильтр» создана система генерации адресов:

```
F2: =CEILING.MATH(COUNT(Автомаркировка!G:G)+10,100)  ← размер данных
F1: =CONCAT("$F$4:","$F$",F2)  ← "$F$4:$F$1400"
M1: =CONCAT("$M$4:","$M$",F2)  ← "$M$4:$M$1400"
... и ещё 13 таких же (B2, C2, E2, G1, I1, J1, K1, L1, N1, O2, P2, U1, AK1)
```

Эти адреса используются spill-формулами через `INDIRECT()` для автоматического подстраивания под объём данных.

## 8. Рекомендации

### 8.1 Настройки Excel (немедленно)

| Настройка | Где | Что сделать |
|-----------|-----|-------------|
| Ручной пересчёт | Файл → Параметры → Формулы | ✅ Уже стоит |
| **Пересчёт перед сохранением** | Там же | ❌ **СНЯТЬ галочку** |
| Фоновый пересчёт | Там же | ❌ Отключить |

**Эффект:** сохранение ускорится на 30-60 секунд.

### 8.2 Быстрые улучшения в формулах

| Где | Что | Эффект |
|-----|-----|--------|
| Автомаркировка FG, FH | Заменить OFFSET на SCAN | Убирает «самоссылку» |
| Автомаркировка BJ:BW | Сократить с 19 до 8 | -33K формул |
| Автомаркировка CU:DL | Сократить с 16 до 6 | -30K формул |
| Автомаркировка FV:FZ | Удалить (дубликат FP:FZ) | -15K формул |
| Расчет HD (50 формул) | Удалить | -50 формул |
| Все INDEX/MATCH | Заменить на XLOOKUP | В 2-3 раза быстрее |

### 8.3 Процесс работы

**Текущий:**
```
Прибор → edit.xlsx → Обработка.xlsx → Обработка БФ.xlsx → Юнтолово...xlsx
```

**Рекомендуемый:**
1. **Создать шаблон «Обработка_шаблон.xlsx»** — без данных, только формулы
2. **Для нового объекта:** копия шаблона → импорт данных → сохранить как «Обработка_[Объект].xlsx»
3. **Тестовый лист:** создать лист «Тест» с 5 строками → проверять формулы там перед вставкой в основной файл

### 8.4 Тестирование формул

Перед изменением формулы в основном файле:
1. Скопировать формулу на лист «Тест» (5 строк)
2. Изменить на «Тесте» → проверить результат
3. Если ок — найти все копии через `formula_search.py` → заменить
4. F9 → проверить что не сломалось

### 8.5 Инструменты поиска и замены

| Файл | Назначение |
|------|------------|
| `formula_search.py` | Поиск, разбор и замена формул |
| `full_analysis.py` | Полный анализ файла (без Excel) |
| `analyze_open_excel.py` | Анализ открытого Excel |
| `find_spill_com.py` | Поиск spill-формул через Excel COM |
| `analyze_automark.py` | Анализ Автомаркировки |

**Использование:** `python formula_search.py`

## 10. Глоссарий

| Термин | Значение |
|--------|----------|
| spill-формула | Формула Excel 365, которая «разливается» на соседние ячейки |
| ДВССЫЛ / INDIRECT | Превращает текст адреса в ссылку |
| INDEX/MATCH | Аналог VLOOKUP — поиск по таблице |
| TRANSPOSE | Транспонирует строки в столбцы и обратно |
| TEXTSPLIT | Разбивает строку по разделителю (Excel 365) |
| FILTER | Фильтрует диапазон по условию (Excel 365) |
| UNIQUE | Убирает дубликаты (Excel 365) |
| SORT / SORTBY | Сортировка (Excel 365) |
| CONCAT | Склеивает текст |
| LET | Определяет переменные в формуле (Excel 365) |
| BYCOL | Применяет функцию к каждому столбцу (Excel 365) |
| TOCOL / TOROW | Преобразует в столбец/строку (Excel 365) |
