# Анализ Excel-конвейера обработки замеров

## Структура папки

```
excel_analysis/
├── README.md                          ← Этот файл
├── 01_Автомаркировка_документация.md  ← Полная документация листа Автомаркировка
├── 02_Архитектура_Обработка.md        ← Карта всех 17 листов, зависимости, spill-формулы
├── 03_Рекомендации.md                 ← Приоритизированные рекомендации по улучшению
├── full_analysis.json                 ← Полный анализ формул (openpyxl)
├── formula_activity.json              ← Какие формулы работают, какие пустые
├── open_excel_analysis.json           ← Анализ открытого Excel (COM)
├── all_formulas.txt                   ← Все уникальные формулы
└── dependencies.dot                   ← Граф зависимостей листов (Graphviz)
```

## Краткое содержание

### 01_Автомаркировка_документация.md
- Полная карта всех 191 столбца
- Зависимости между столбцами
- Какие столбцы активны, какие пусты на текущем объекте
- Логика парсинга, маркировки, примечаний
- Именованные диапазоны (справочники)
- Потенциальные проблемы

### 02_Архитектура_Обработка.md
- Карта всего конвейера: от прибора до выходного файла
- Статистика по каждому листу
- Spill-формулы (динамические, авто-растягивающиеся)
- Динамические диапазоны (CONCAT → INDIRECT)
- Внешние ссылки на источник
- Мёртвые зоны
- Глоссарий

### 03_Рекомендации.md
- **Приоритет 1:** Настройки Excel, шаблон, тестовый лист
- **Приоритет 2:** XLOOKUP вместо INDEX/MATCH, убрать OFFSET, сократить столбцы
- **Приоритет 3:** Документация, карточка объекта, spill-блоки
- **Приоритет 4:** Модульный файл, Python-конвейер
- Чек-лист перед новым объектом
