## 🎯 Принципы проектирования

### SOLID

| Принцип | Применение |
|---------|------------|
| **S**ingle Responsibility | Один класс — одна функция |
| **O**pen/Closed | Открыто для расширения, закрыто для изменений |
| **L**iskov Substitution | Подклассы заменяют базовый класс |
| **I**nterface Segregation | Много специализированных интерфейсов |
| **D**ependency Inversion | Зависимость от абстракций |

### DRY

**Don't Repeat Yourself** — если код повторяется 3 раза, выноси в функцию.

### KISS

**Keep It Simple, Stupid** — простое решение лучше сложного.

### YAGNI

**You Aren't Gonna Need It** — не делай функциональность "на будущее".

## 📏 Метрики качества

| Метрика | Цель | Минимум | Критично |
|---------|------|---------|----------|
| **Покрытие тестами** | >90% | >80% | <50% |
| **Строк на файл** | 250-350 | 200-400 | >500 |
| **Строк на класс** | 150-200 | 100-250 | >300 |
| **Строк на функцию** | 20-40 | 10-50 | >100 |
| **Параметров на функцию** | ≤4 | ≤5 | >7 |
| **Сложность (cyclomatic)** | ≤8 | ≤10 | >20 |
| **Время тестов** | <30 сек | <60 сек | >5 мин |
| **Время сборки** | <30 сек | <60 сек | >5 мин |

## 🔧 Конфигурации инструментов

### ruff.toml

```toml
line-length = 100
target-version = "py310"

[lint]
select = ["E", "F", "W", "I", "N", "UP", "B", "C4", "SIM"]
```

### pytest.ini

```ini
[pytest]
testpaths = tests
python_files = test_*.py
python_functions = test_*
addopts = -v --cov=src --cov-report=html --cov-fail-under=80
```

### mypy.ini

```ini
[mypy]
python_version = 3.10
strict = true
warn_return_any = true
warn_unused_ignores = true
```
