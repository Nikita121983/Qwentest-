## 2. РАЗМЕРЫ ПРОЁМОВ

### 2.1. Минимальные размеры проёма

| Параметр | Минимум (мм) | Норма |
|----------|--------------|-------|
| **Ширина** | 600 | СНиП 21-01-97 |
| **Высота** | 1800 | СНиП 21-01-97 |

### 2.2. Рекомендуемые размеры проёма

**Для двери 800×2000 мм:**
```
Ширина проёма = 880-900 мм (полотно + монтажный зазор × 2)
Высота проёма = 2050-2070 мм (полотно + монтажный зазор + порог)
```

**Формула:**
```
Ширина проёма = Ширина полотна + (Монтажный зазор × 2)
Высота проёма = Высота полотна + Монтажный зазор + Порог
```

**Монтажный зазор:** 20-40 мм (из `abbreviations.json`)

### 2.3. Допуски на проём

Из `abbreviations.json`:

```json
{
  "tolerance": {
    "60": "Допуск ширины 60 мм",
    "90": "Допуск ширины 90 мм",
    "200": "Допуск высоты 200 мм"
  }
}
```

**Применение:**
- **Допуск ширины 60 мм** — для межкомнатных дверей
- **Допуск ширины 90 мм** — для входных дверей
- **Допуск высоты 200 мм** — для всех типов дверей

## 3. ПРОВЕРКА РАЗМЕРОВ (ИЗ ЧАТОВ)

### 3.1. Эвристики из чатов

**Ширина:**
| Значение | Интерпретация |
|----------|---------------|
| **800-1200 мм** | Стандартная дверь |
| **1400-2000 мм** | Двустворчатая дверь |
| **< 600 мм** | ❌ Подозрительно мало (возможно окно/люк) |
| **> 2000 мм** | ⚠️ Нестандарт (панорамная) |

**Высота:**
| Значение | Интерпретация |
|----------|---------------|
| **2000-2300 мм** | Стандартная высота |
| **> 2500 мм** | ⚠️ Панорамная дверь |
| **< 1800 мм** | ❌ Подозрительно мало (возможно окно/люк) |

### 3.2. Алгоритм валидации (из 10_MEASUREMENT_ALGORITHMS.md)

```kotlin
fun validateDoorSize(width: Double, height: Double): ValidationResult {
    val errors = mutableListOf<String>()
    val warnings = mutableListOf<String>()

    // Проверка ширины
    if (width < 600) {
        errors.add("Ширина ${width} мм < 600 мм (минимум)")
    } else if (width > 2000) {
        warnings.add("Ширина ${width} мм > 2000 мм (нестандарт)")
    }

    // Проверка высоты
    if (height < 1800) {
        errors.add("Высота ${height} мм < 1800 мм (минимум)")
    } else if (height > 2500) {
        warnings.add("Высота ${height} мм > 2500 мм (нестандарт)")
    }

    return ValidationResult(errors, warnings)
}
```

## 5. РАСЧЁТ РЕКОМЕНДУЕМЫХ РАЗМЕРОВ

### 5.1. Формула

```kotlin
data class DoorRecommendation(
    val width: Double,      // Рекомендуемая ширина полотна
    val height: Double,     // Рекомендуемая высота полотна
    val mountingGap: Double,// Монтажный зазор
    val notes: List<String>
)

fun calculateDoorRecommendation(
    minWidth: Double,  // Минимальная ширина из 3 замеров
    minHeight: Double  // Минимальная высота из 3 замеров
): DoorRecommendation {
    // Монтажный зазор (из словаря: 20-40 мм)
    val mountingGap = 30.0 // среднее значение

    // Рекомендуемые размеры
    val recommendedWidth = minWidth - mountingGap * 2
    val recommendedHeight = minHeight - mountingGap

    // Округление до стандартного размера
    val standardWidth = roundToStandard(recommendedWidth, DoorWidths)
    val standardHeight = roundToStandard(recommendedHeight, DoorHeights)

    return DoorRecommendation(
        width = standardWidth,
        height = standardHeight,
        mountingGap = mountingGap,
        notes = emptyList()
    )
}

// Стандартные размеры полотен
val DoorWidths = listOf(600, 700, 800, 900, 1000, 1100, 1200, 1400, 1600, 1800)
val DoorHeights = listOf(1900, 2000, 2100, 2200, 2300, 2400, 2500)

fun roundToStandard(value: Double, standards: List<Int>): Double {
    // Находим ближайший стандартный размер
    return standards.minByOrNull { kotlin.math.abs(it - value) }?.toDouble() ?: value
}
```

### 5.2. Пример расчёта

**Входные данные:**
```
Минимальная ширина: 999.3 мм
Минимальная высота: 2208.4 мм
```

**Расчёт:**
```
Ширина полотна = 999.3 - 30×2 = 939.3 мм → округляем до 900 мм (стандарт)
Высота полотна = 2208.4 - 30 = 2178.4 мм → округляем до 2100 мм (стандарт)
```

**Результат:**
```
✅ Рекомендуемая дверь: 900 × 2100 мм
✅ Монтажный зазор: 30 мм
```
