## Выводы для нашего приложения

### 1. Пользователь УЖЕ подписывает замеры
- Короткие имена: "Дп", "Дст", "Дствп"
- Полные маркировки: "31 П1.1.Кв1 Ст-6.1/15.1..."
- **Нужно:** поддержка шаблонов имён

### 2. Группировка по объектам
- Этаж → Квартира → Стена → Замер
- **Нужно:** иерархическая структура

### 3. Повторные замеры
- Одно и то же измеряется 2-3 раза
- **Нужно:** группировка повторных замеров

### 4. Реальное время замера
- 18:55 → 19:22 — замеры в разное время
- **Нужно:** точный timestamp (не экспорта!)

### 5. Быстрый ввод имён
- Пользователь вводит "Дп", "Дст" вручную
- **Нужно:** кнопки быстрого ввода + история имён
