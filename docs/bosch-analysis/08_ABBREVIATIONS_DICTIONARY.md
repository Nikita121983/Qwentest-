## 📋 ПОДТВЕРЖДЁННЫЕ СОКРАЩЕНИЯ (основные)

### Стяжка
| Код | Значение |
|-----|----------|
| **ст-** | Нет стяжки (перед проёмом) |
| **ст+** | Есть стяжка (за проёмом) |
| **ст-6.1/15.1** | Нет стяжки перед (-6.1), есть за (+15.1) |

### Наличник
| Код | Значение |
|-----|----------|
| **сн** | С наличником |
| **бн** | Без наличника |

### Открывание
| Код | Значение |
|-----|----------|
| **пр** | Правое открывание |
| **лв** | Левое открывание |

### Стены
| Код | Значение |
|-----|----------|
| **скц** | Камень стеновой |
| **мнт** | Монолит |
| **крп** | Кирпич |
| **гб** | Газобетон |

### Штукатурка
| Код | Значение |
|-----|----------|
| **шт-** | Стены не оштукатурены |
| **шт+** | Стены оштукатурены |
