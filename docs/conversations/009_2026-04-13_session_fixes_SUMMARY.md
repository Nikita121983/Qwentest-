# Резюме сессии: Исправление протокола GLM и критических багов

**Дата:** 13 апреля 2026 г.
**Версия:** v1.0.22

## Основные проблемы
1.  **Нестабильная связь:** Рулетка не реагировала на команды.
    *   **Причина:** Ошибочная формула байта заголовка EDC. Считали `syncControl` старшим битом (0x40), а он младший (0x01).
    *   **Решение:** Исправлена формула на `(devMode << 2) | (keypadBypass << 1) | syncControl`.
2.  **Кубики не переключали режимы:** Приложение отправляло UI-индексы вместо EDC-значений.
    *   **Решение:** Внедрен маппинг `turnSyncModeToEDCMode` (например, Площадь UI 2 -> EDC 4).
3.  **Кнопка "Включить" (Лазер):** Не работала или работала некорректно.
    *   **Решение:** Реализована логика "тумблера" (Laser On -> Trigger).
4.  **Зависание интерфейса:** После замера кнопки переставали работать.
    *   **Причина:** `SendThread` умирал при таймауте (`break` вместо `continue`).
    *   **Решение:** Поток теперь продолжает работу после таймаута.
