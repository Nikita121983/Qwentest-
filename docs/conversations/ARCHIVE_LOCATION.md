# Выбор места хранения архива диалогов

Этот файл определяет, где хранится архив диалогов проекта.

---

## Текущая настройка

**Место хранения:** `LOCAL` (локально в проекте)  
**Путь:** `docs/conversations/`

---

## Доступные опции

### Опция 1: LOCAL (локально в проекте)

```
{Project}/
└── docs/
    └── conversations/
        ├── INDEX.md
        ├── 001_..._FULL.md
        └── 001_..._SUMMARY.md
```

**Плюсы:**
- ✅ Архив рядом с кодом проекта
- ✅ Легко найти при работе с проектом
- ✅ Проект самодостаточен

**Минусы:**
- ⚠️ Архивы разбросаны по разным проектам
- ⚠️ Сложнее искать по всем проектам сразу

---

### Опция 2: GLOBAL (глобально в ~/.qwen)

```
~/.qwen/
└── conversations/
    └── {Project}/
        ├── INDEX.md
        ├── 001_..._FULL.md
        └── 001_..._SUMMARY.md
```

**Плюсы:**
- ✅ Все архивы в одном месте
- ✅ Легко искать по всем проектам
- ✅ Глобальный индекс

**Минусы:**
- ⚠️ Архив отдельно от проекта
- ⚠️ Нужно помнить путь

---

## Как изменить место хранения

### Из LOCAL в GLOBAL

1. Создай папку: `~/.qwen/conversations/{ProjectName}/`
2. Перемести файлы из `docs/conversations/` в новую папку
3. Измени настройку выше на `GLOBAL`
4. Обнови `docs/conversations/INDEX.md` — добавь ссылку на глобальную папку
5. Обнови глобальный индекс `~/.qwen/conversations/INDEX.md`

### Из GLOBAL в LOCAL

1. Создай папку: `docs/conversations/` в проекте
2. Перемести файлы из `~/.qwen/conversations/{ProjectName}/` в локальную папку
3. Измени настройку выше на `LOCAL`
4. Обнови глобальный индекс `~/.qwen/conversations/INDEX.md`

---

## Глобальный индекс

Независимо от выбора места хранения, проект должен быть зарегистрирован в глобальном индексе:

**Путь:** `~/.qwen/conversations/INDEX.md`

---

*Этот файл создан автоматически. Измени настройку при необходимости.*
