## 1. ФОРМАТ МАРКИРОВКИ

### 1.1. Полный формат

**Пример:** `31 П1.1.Кв1 Ст-6.1/15.1 Пр Сн Скц/Мнт`

| Часть | Код | Значение |
|-------|-----|----------|
| **31** | Номер двери | Дверь №31 |
| **П1.1** | Подъезд.Секция | Подъезд 1, Секция 1 |
| **Кв1** | Квартира | Квартира №1 |
| **Ст-6.1/15.1** | Стяжка | Нет стяжки перед (-6.1), есть за (+15.1) |
| **Пр** | Открывание | Правое |
| **Сн** | Наличник | С наличником |
| **Скц/Мнт** | Стены | Слева камень, Справа монолит |

### 1.2. Структура CSV (Bosch)

| Колонка | Название | Пример | Использование |
|---------|----------|--------|---------------|
| **A** | Название | `31 П1.1.Кв1...` | Парсить маркировку |
| **B** | Значение | `2188,6` | Размер в мм |
| **C** | Ед. изм | `mm` | Единицы измерения |
| **D** | Дата | `05/03/26` | Дата замера |
| **E** | Время | `19:22` | ⚠️ Время экспорта (не использовать!) |
| **F** | Вид | `Длина` | Тип замера |

## 3. ГРУППИРОВКА ЗАМЕРОВ

### 3.1. На 1 дверь = 6 замеров

```
31 П1.1.Кв1 Ст-6.1/15.1 Пр Сн Скц/Мнт
├─ Замер 1: Ширина (верх)    — 999,3 mm
├─ Замер 2: Ширина (середина) — 1010,6 mm
├─ Замер 3: Ширина (низ)     — 1014,5 mm (с маркировкой)
├─ Замер 4: Высота (левая)   — 2208,4 mm (Дп)
├─ Замер 5: Высота (середина) — 2227,1 mm
└─ Замер 6: Высота (правая)  — 2228,9 mm
```

### 3.2. В CSV

**Маркировка только на последнем замере в группе!**

```csv
Измерение;2228,9;mm;05/03/26;19:22;Длина  ← Высота правая
Измерение;2227,1;mm;05/03/26;19:22;Длина  ← Высота средняя
Дп;2208,4;mm;05/03/26;19:22;Длина         ← Высота левая (До плиты)
Измерение;999,3;mm;05/03/26;19:22;Длина   ← Ширина верх
Измерение;1010,6;mm;05/03/26;19:22;Длина  ← Ширина середина
31 П1.1.Кв1...;1014,5;mm;05/03/26;19:22;Длина ← Ширина низ (с маркировкой)
```

## 4. АЛГОРИТМ ПАРСИНГА

### 4.1. Регулярное выражение

```regex
(\d+)                         # Номер двери
\s*
(П\d+\.\d+)                   # Подъезд.Секция
\.
(Кв\d+)                       # Квартира
\s*
(Ст-?\d+\.?\d*/\d+\.?\d*)?    # Стяжка (опционально)
\s*
(Пр|Лв)?                      # Открывание (опционально)
\s*
(Сн|Бн|Сн\w+|Бн\w+)?          # Наличник (опционально, 100+ вариантов)
\s*
(\w+/\w+)?                    # Стены (опционально)
\s*
(Шт-?\w+)?                    # Штукатурка (опционально, 30+ вариантов)
```

## 5. ИМПОРТ В MEASURINGASSISTANT

### 5.1. Алгоритм

1. **Прочитать файл** (с авто-маркировками)
2. **Перевернуть порядок** (снизу вверх — от старых к новым)
3. **Найти строки с маркировками** (парсинг по regex)
4. **Сгруппировать по 6 замеров на дверь:**
   - Найти строку с маркировкой
   - Взять предыдущие 5 строк
   - Сгруппировать в 1 объект (6 точек замера)
5. **Для каждой группы:**
   - `object_id` = из маркировки (D_31_Кв1)
   - `measurement_id` = новый (m_001, m_002...)
   - 6 точек: p1-p6 (ширина 3 + высота 3)
   - Распарсить маркировку в поля объекта
6. **Сохранить** с пометкой `imported_from: "Bosch MM"`

### 5.2. Предупреждение пользователю

```
⚠️ Время замеров недоступно (только дата экспорта)
⚠️ Порядок замеров восстановлен (обратная сортировка)
⚠️ Найдено 185 дверей из 1582 замеров (~6 замеров на дверь)
```
