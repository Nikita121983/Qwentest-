# Приложение для замеров — Краткий бриф

**Версия:** 1.0
**Дата:** 2 апреля 2026 г.
**Статус:** Черновик

---

## 📱 Платформы

| Платформа | Технология | Приоритет |
|-----------|------------|-----------|
| **Android** | Kotlin (Native) | 🔴 Высокий |
| **Windows** | C# / WPF или Avalonia | 🟡 Средний |

---

## 🔧 Ключевые решения

| Решение | Выбор |
|---------|-------|
| **Распознавание речи** | On-device (Whisper/Vosk) |
| **PDF просмотр** | Интеграция с Xodo (Intent) |
| **Синхронизация** | Нет (оффлайн работа) |
| **Хранение** | Папка проекта + SQLite |

---

## 🎯 Назначение

Профессиональный инструмент для замерщиков дверей/окон с:
- Подключением к лазерной рулетке (Bosch GLM 50 C)
- Голосовыми заметками с оффлайн-распознаванием
- Привязкой к PDF-планам
- Экспортом в нормальные форматы (XLSX/CSV)
