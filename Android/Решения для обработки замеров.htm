❗ Почему у тебя “пропадают замеры”

Это не баг (или не только баг), это архитектура:

1. Список = временное хранилище

  • значения живут в UI слое

  • не гарантировано сохраняются как отдельные записи

2. Автосохранение только проекта

  • сохраняется структура (стены, объекты)

  • а не “сырой список измерений” MeasureOn

3. Потеря может происходить из-за:

  • перепривязки к объектам

  • очистки списка

  • багов синхронизации (облако)

  • лимитов/перезаписи (не документировано, но есть косвенные признаки)
